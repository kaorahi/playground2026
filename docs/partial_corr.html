偏相関係数の教材

「第三の変数の影響を取り除いたあと、2つの変数がどれくらい一緒に動くか」を、数式とインタラクティブなデモで確認します。

1. 偏相関係数とは

偏相関係数は、変数 X と Y の関係から、共通して影響している変数 Z の効果を取り除いた相関です。

通常の相関は「X と Y が一緒に動くか」を見ます。偏相関は「Z で説明できる部分を除いても、X と Y が一緒に動くか」を見ます。

例

アイスの売上 X と熱中症搬送数 Y には正の相関があるかもしれません。しかし、どちらも気温 Z の影響を受けます。

このとき偏相関は、「気温の影響をそろえたうえでも、アイスの売上と熱中症搬送数に関係が残るか」を調べます。

見方

1 に近い Z を除いても X と Y は正に関係します。

0 に近い 関係はほとんど残りません。

-1 に近い Z を除くと X と Y は逆向きに関係します。

2. 計算式

3変数 X, Y, Z の偏相関係数は、通常の相関係数から次の式で計算できます。

rXY·Z = (rXY - rXZrYZ) / sqrt((1 - rXZ2)(1 - rYZ2))

rXY は X と Y の相関、rXZ は X と Z の相関、rYZ は Y と Z の相関です。

3. 残差で理解する

手順 1

X を Z で予測し、予測できなかった残りを取り出します。

手順 2

Y も Z で予測し、予測できなかった残りを取り出します。

手順 3

その2つの残り同士の相関を計算します。これが偏相関です。

つまり偏相関は、「Z で説明できない X の部分」と「Z で説明できない Y の部分」の相関です。

4. インタラクティブ・デモ

下のスライダーを動かすと、X と Y の見かけの相関、Z との関係、Z を取り除いたあとの偏相関が変わります。

設定

通常の相関 rXY
X と Z の相関 rXZ
偏相関 rXY·Z

X と Y の散布図 Z の値: 低 高
Z を取り除いた残差同士 Z の値: 低 高

5. デモの読み取り方

通常の相関が大きいのに、偏相関が小さい場合
X と Y の関係は、主に Z の影響で生じている可能性があります。

通常の相関も偏相関も大きい場合
Z を除いても X と Y の関係が残っています。

通常の相関は小さいのに、偏相関が大きい場合
Z の影響が X-Y の関係を隠していた可能性があります。

偏相関が因果を示すわけではありません
偏相関は統計的な関係を表す指標であり、因果関係の証明ではありません。

6. 練習問題

問題 1

rXY = 0.60, rXZ = 0.70, rYZ = 0.70 のとき、偏相関は通常の相関より大きくなりますか、小さくなりますか。

答え: 小さくなります。X と Y の見かけの相関の一部が Z によって説明されるためです。


問題 2

偏相関が 0 に近い場合、どのように解釈できますか。

答え: Z の影響を取り除くと、X と Y の線形な関係はほとんど残らないと解釈できます。


問題 3

偏相関だけで「X が Y の原因である」と言えますか。

答え: 言えません。偏相関は因果の証明ではなく、制御した変数の選び方にも依存します。
